## 📝 TEST: ¿Cuál es la diferencia entre un bucle Mientras y un bucle Para?
- **Tipo:** assistant
- **Tiempo:** 8221ms
- **PSeInt:** No
- **Herramienta:** No

### 💬 Respuesta Generada:
```
Un bucle **Mientras** se ejecuta mientras una condición específica sea verdadera. Se usa cuando no se conoce de antemano cuántas veces se necesitan iteraciones. Por ejemplo: *Mientras X < 10: imprimir X*.

Un bucle **Para** itera sobre una secuencia o colección (como un array o lista). Se usa cuando se sabe de antemano el número de elementos. Por ejemplo: *Para i en 0 a 9: imprimir i*.

La clave está en la condición: el bucle Mientras continúa mientras algo cierto, y el bucle Para repite para cada elemento.
```

- **Tipo:** assistant
- **Tiempo:** 12636ms
- **PSeInt:** Sí
- **Herramienta:** No

### 💬 Respuesta Generada:
```
Una función recursiva es una función que se llama a sí misma durante su ejecución. Para definirla correctamente, primero se establece un caso base (condición de terminación) y luego se define el paso recursivo que reduce el problema hasta alcanzarlo.
```
