# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-04-23

### Added
- Initial project structure using Tauri v2 and Vite + TypeScript.
- Modular visual system "Tablero con Bloques" with draggable panels.
- Robust theming system with Obsidian (dark) and Nordic (light) themes.
- Truly Infinite Grid background using CSS variables and math-based panning.
- Mouse-centered zoom engine for high-precision navigation.
- Dynamic Relationship Engine: Curved Bézier links that update in real-time.
- Interactive Block Factory: Users can now spawn and link blocks dynamically.
- SOLID/POO refactor: Decoupled Viewport (State), ViewportController (Input), and UIComponent (Base class).
- Component-based architecture for UI blocks.
- Basic project documentation (Roadmap, Requirements, Visual System).

### Fixed
- HTML entity parsing error in pseudocode preview.
- Asset loading paths for Tauri development.
