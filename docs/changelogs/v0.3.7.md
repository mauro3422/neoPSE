# Changelog NeoPSE - v0.3.7 "Stable Interaction & UI Polish"

Cierre de la fase de endurecimiento del motor de interacción y corrección de bugs visuales críticos.

## 🚀 Mejoras y Bugfixes
- **Fix de Interacción:** Corregido el bug de IDs huérfanos que impedía que los bloques nuevos fueran arrastrables o enlazables.
- **Scrollbar Pro:** Implementación de sliders personalizados (thin, dark) para una estética de IDE premium.
- **Auto-Resize de Notas:** Las notas ahora ajustan su altura dinámicamente al contenido.
- **Utilidades Centralizadas:** Creación de `DOMUtils` para gestionar comportamientos de UI de forma DRY.

## 🛠️ Refactorización
- **spawnBlockInstance:** Ahora captura y sincroniza correctamente los IDs generados por la Factory.
- **SpecializedBlocks:** Delegación del auto-ajuste a utilidades externas, reduciendo la complejidad de clase.
