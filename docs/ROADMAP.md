# NeoPSE Roadmap

Este documento detalla la evolución técnica y funcional de NeoPSE.

## Fase 1: Motor y Estructura (COMPLETADA ✅)
- [x] Viewport infinito con zoom y pan.
- [x] Registro de bloques modular (SOLID).
- [x] Motor de enlaces con curvas de Bézier.
- [x] Persistencia de estado y rehidratación.
- [x] Asistente IA persistente en la interfaz.

## Fase 2: Interacción Avanzada (EN PROGRESO 🚧)
- [x] Cable Fantasma (Ghost Linking) con previsualización.
- [x] Anclaje dinámico a bordes.
- [ ] **Menú Contextual:** Click derecho para invocar el registro de bloques.
- [ ] **Selección Múltiple:** Arrastrar para seleccionar varios bloques.
- [ ] **Minimización de Bloques:** Colapsar contenido para ahorrar espacio.

## Fase 3: El Intérprete (PRÓXIMAMENTE 🧠)
- [ ] **Parser de Pseudocódigo:** Conversor de texto a AST (Abstract Syntax Tree).
- [ ] **Motor de Ejecución:** Recorrido de grafos y ejecución lógica de bloques enlazados.
- [ ] **Gestión de Variables:** Panel para ver el estado de la memoria durante la ejecución.

## Fase 4: Ecosistema y Publicación
- [ ] **FileTree:** Sistema de archivos virtuales.
- [ ] **Exportación:** Guardar como archivo `.pse` o exportar a código real.
