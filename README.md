# NeoPSE - Motor de IDE de Pseudocódigo Modular

NeoPSE es un entorno de desarrollo integrado (IDE) basado en nodos y flujo visual, diseñado para la enseñanza y el desarrollo de algoritmos en pseudocódigo.

## 🚀 Características Actuales
- **Espacio Infinito:** Navegación fluida con Paneo y Zoom (Viewport Inteligente).
- **Arquitectura Data-Driven:** Sistema de componentes basado en registros (SOLID/DRY).
- **Enlaces Inteligentes:** Conexiones de Bézier dinámicas con anclaje automático a bordes y previsualización de "Cable Fantasma".
- **Persistencia Total:** Guardado automático y rehidratación de workspace, historial de chat de IA y preferencias de tema.
- **Asistente IA Integrado:** Panel de ayuda contextual persistente.

## 🛠️ Tecnología
- **Core:** TypeScript + Vanilla CSS.
- **Motor Gráfico:** SVG Dinámico + Transformaciones de Matriz.
- **Estado:** WorkspaceState con Debounced Persistence.
